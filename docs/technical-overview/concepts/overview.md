# Concepts

Key concepts and paradigms in ODE.

## Offline-First

ODE is designed to work offline-first, allowing data collection even without internet connectivity.

[Learn more about Offline-First](/docs/technical-overview/concepts/offline-first)

## JSON Forms

ODE uses JSON Forms for flexible form rendering and customization.

[Learn more about JSON Forms](/docs/technical-overview/concepts/json-forms)

## App Bundles

Applications are packaged as bundles containing forms, schemas, and custom code.

[Learn more about App Bundles](/docs/technical-overview/concepts/app-bundles)

## Custom Applications

ODE supports custom web applications that integrate with Formulus.

[Learn more about Custom Applications](/docs/technical-overview/concepts/custom-apps)
