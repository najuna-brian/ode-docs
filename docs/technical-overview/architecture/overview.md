# Architecture Overview

Overview of ODE components and their interactions.

## System Components

ODE consists of several key components:

- **Formulus**: Mobile application for data collection
- **Synkronus**: Server backend for synchronization and data management
- **Formplayer**: Web-based form renderer
- **Synkronus CLI**: Command-line interface for server management
